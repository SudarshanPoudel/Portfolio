Screenscore is a sleek, responsive frontend project designed for movie enthusiasts. It utilizes the TMDB API to fetch real-time movie data, offering users a feature-rich platform to explore and manage their favorite films.

    Features :
  1. Optimized for all devices with a focus on great UI/UX.
  2. Always updated with the latest movies and data from TMDB.
  3. Choose from multiple themes to personalize the browsing experience.
